## 🚀 Pour lancer le projet :

- npm install
- npm start

## ✨ Comment ça marche ?

- Entrez votre nom et l'id de votre room

- Commande disponible dans le chat : *"/settime heure"*
  
  Ex: "/settime 14" -> change la date du RDV à 14h
